SETUP (ONE TIME ONLY)
================================================================================

STEP 1: Run SQL Setup
----------------------
1. Open phpMyAdmin: http://localhost/phpmyadmin
2. Select database: admin_placement_db
3. Click "SQL" tab
4. Copy contents from: sql/create_admin_notifications.sql
5. Paste and click "Go"

That's it! No other setup needed.

USING THE FEATURES
================================================================================

FEATURE 1: Customize Application Forms
---------------------------------------
1. Go to any drive â†’ Click "Edit Drive"
2. Click "Customize Application Form" button (top-right)
3. Check/uncheck fields you want to include
4. Add custom fields if needed
5. Click "Save Form Configuration"
6. Preview the form to verify

Example use case:
- Company doesn't need PG details? Uncheck all PG fields
- Company wants portfolio link? Add custom field "Portfolio URL"
- Only need basic info? Deselect advanced fields

FEATURE 2: Deadline Notifications
----------------------------------
1. Look for bell icon in sidebar (below logo)
2. Red badge shows unread notification count
3. Click bell to view all notifications

Notifications are created automatically when:
- Deadline expires (1+ hour after deadline)
- Deadline approaching (within 24 hours)

Just refresh any admin page to check for new notifications!
